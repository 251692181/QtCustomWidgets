# QtCustomWidgets
一些Qt的自定义控件，从网上或参照移动端的一些好看的控件样式写成的Qt自定义控件<br>
## 使用方法
目前控件没有提供Qt设计师界面，需要项目中包含源码。详细使用方法可以参考本项目例子
## 更新日志
后续对控件的修改和新增控件会在日志中体现，详细内容请参考[VersionLog](https://github.com/LnJan/WechatLuckyMoneyGetter/blob/master/VERSIONLOG.md)
